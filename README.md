# 기능
- Spacebar로 내 캐릭터를 점프할 수 있음
- 정해진 타임에 따라서 방해요소들이 내 캐릭터 쪽으로 다가옴
- 유저가 캐릭터를 적절한 타이밍에 맞게 방해요소들을 피해 점프해야 게임을 이어갈 수 있음
- 점수가 올라가는데 방해요소들에 부딪혀서 게임이 끝날 때까지 점수를 측정(~ing)
- 점수 랭킹 시스템(희망)
